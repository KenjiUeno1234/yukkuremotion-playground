## 🔍 MinimalTestの仕組み

MinimalTestは既存の動作しているコードと**完全に同じ方法**で画像を読み込みます：

```typescript
// 既存のYukkuriVideo.tsxと全く同じ方法
import { staticFile } from 'remotion';
<Img src={staticFile('image/yukkurilogo.png')} />  // 既存の画像
<Img src={staticFile('slide/S001.png')} />         // 新しい画像
```

このテストにより、以下が判明します：

1. **Remotionの画像読み込み機能が正常か**
2. **既存の画像と新しい画像で違いがあるか**
3. **問題がファイル自体にあるか、コンポーネントのロジックにあるか**
