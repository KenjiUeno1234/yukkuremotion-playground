# 目的

このプロジェクトはRemotionを使用したゆっくり実況動画作成システムです。

# 使用ツール

## MCP (Model Context Protocol)

Remotionを利用した開発では、以下のMCPサーバーを活用してください：

### pixabay-mcp Docs

Remotionの公式ドキュメント、API仕様、ベストプラクティスを参照する際は、**pixabay-mcp Docs**を利用してください。

**利用方法：**
```
mcp__pixabay-mcp-docs__fetch_pixabay_mcp_documentation
mcp__pixabay-mcp-docs__search_pixabay_mcp_documentation
```

**利用シーン：**
- Remotionのコンポーネント、API、フックの使い方を調べる
- 動画のレンダリング、エンコーディング設定を確認する
- ベストプラクティスやパフォーマンス最適化の方法を学ぶ
- エラーのトラブルシューティング

### その他のMCPツール

- **script**: TypeScript/JavaScriptスクリプトの実行
- **Context7**: ライブラリのドキュメント検索（React、TypeScriptなど）
- **Tavily**: Web検索、技術情報の収集
